UNIPROTtoGO(dico, output, pos, dicoGO, GOfile)

source code 

Make ID mapping from Uniprot identifiers to GO identifiers requesting Online(Quick-GO database) of Offline(EBI-GO_databases.tsv)

Parameters:
  • dico (dictionnary) - dictionnary used in OFFLINE requests --> keys = Uniprot identifiers , values = GO identifiers list
  • output (.txt file) - Universe of GO identifiers (univers.txt)
  • pos (int) - positions requested if -pos argument was used
  • dicoGO (dictionnary) - dictionnary used in ONLINE requests --> keys = Uniprot identifiers , values = GO identifiers list
  • GOfile (.tsv file) - Storage of uniprot identifiers to GO identifiers (EBI-GO_databases.tsv)
