Module Hierarchy

  • BAMmaker (script): Make a BAM filtered and sorted from FASTQ and a reference FASTA File.
  • SNP_INDEL_merge (script): Merge SNPs VCF file and InDels VCF file.
  • VARCall (script): Do the SNP/Indel calling from several FASTQ files
  • VCFilter (script): Filter VCF file by deleting lines with variant only in reference, miss informations in alignment and delete INDEL when they are SNP at the same position.
  • VCFmaker_INDEL (script): Do the InDel calling from BAM files.
  • VCFmaker_SNP (script): Do the SNP calling from BAM files.
  • VCFtoFASTA (script): Create concatenate variant fasta file from VCF file.
  • VCFtoMATRIX (script): Create distance matrix in tabular file from VCF file.
  • VCFtoPseudoGenome (script): Make pseudo genome from a VCF file and the reference FASTA file (1 contig).
  • reportMaker (script): Make a PDF report from the log file generated by the VarCall workflow.
