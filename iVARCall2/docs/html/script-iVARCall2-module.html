Script iVARCall2

Do the SNP calling from several FASTQ files

Requires:
  • BWA
  • samtools
  • picard-tools
  • Trimmomatic
  • GATK
  • SnpSift
  • BAMmaker, iVCFmaker_SNP, iVCFmerge_SNP, VCFilter, VCFtoMATRIX and VCFtoFASTA in $PATH
  • bgzip
  • tabix
  • vcf-merge
  • Vcf.pm (vcftools)

    Paired-end reads must be nammed as "prefix_R1.XXXX" and "prefix_R2.XXXX". Example : E1_R1.fq and E1_R2.fq
