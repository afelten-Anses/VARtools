Module EveryGO

source code

EveryGO workfow. In order to perform GO-terms enrichment analysis for one or more nodes/comparisons of the phylogeny stored in the XML file.

Requires:
  • python 2.7 (tested with 2.7.6)
  • lxml
  • GOtrimmer_xml
  • GOwalker
  • GoView
